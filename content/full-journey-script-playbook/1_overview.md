# 1.0 Overview

This module is a complete script library for the full Clark Hyundai sales journey.

Use it when you want one place for clean, ready-to-use language from first contact through delivery and follow-up.

## 1.1 Objective
Deliver a consistent guest experience by using clear word tracks that are calm, respectful, and easy to execute.

## 1.2 How to Use This Module
1. Pick the stage you are currently in.
2. Use the exact script first.
3. Personalize only after the structure is natural.
4. Stay concise and avoid over-explaining.

## 1.3 Suggested Practice Flow
- Run stages 1–4 as a continuous opening sequence.
- Run stages 5–9 for presentation-to-numbers transitions.
- Run stages 10–15 as close, delivery, and retention practice.
